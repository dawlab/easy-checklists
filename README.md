# EasyChecklists
App with checklists. You can create many lists with tasks, goals, ingredients etc. 

## What have i learned in this project?

- Create UI programmatically - without storyboards. 
- SnapKit 
- Localizable.strings, Swiftgen 
- Realm CRUD
